==== Architecture Proposal ====

This document presents a proposal of architecture for message signaling in AGL.

Also available as a PDF Document

==== Documentation ====

Developer Guidelines are available as a PDF Document

GPS Binding example is available on Github: github:iotbzh/af-gps-binding

==== OpenXC Demo ====

A reference HTML5 application has been developed: see github:iotbzh/txc-demo.

This application uses a OpenXC trace file to display 4 different panels representing live vehicle data.

It's available as an AGL Application paclage installable through AGL Application Framework.
